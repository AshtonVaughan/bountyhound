🎯 Key Features

  • Parallel Execution: CLI scanning + Browser testing run simultaneously
  • LLM-Powered Discovery: AI generates hypothesis cards from recon data
  • Gap-Triggered Re-discovery: If no findings, refine and retry
  • POC Validation: Every finding verified with curl before reporting
  • Multi-User Support: Test IDOR/BOLA with User A/B credentials
  • Evidence Collection: Screenshots, network logs, request/response captures
