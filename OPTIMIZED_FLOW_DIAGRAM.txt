        ├─ Recon (30 days): Fresh → SKIP Phase 1 entirely

  Phase 0.5: SKIPPED (use cached stack)

    ├─ Input: Cached recon + new endpoints found

    ├─ ffuf against cached+new URLs (check cache first)
    ├─ sqlmap against cached+new endpoints
    ├─ nuclei templates against all (cache hit instantly)
    ├─ ... (only test what's NOT cached)
    └─ Results: Only NEW findings appear

    ├─ Saved 5 min (recon skipped)
    ├─ Saved 10+ min (agent caches hit, no tool execution)
    ├─ Saved 2 min (discovery instant with cached data)
